### A
***ABI***: Application Binary Interface 应用程序二进制接口。ABI中描述了应用程序在系统上运行时必须遵守的变成约定。   
***ACPI***: Advanced Configuration and Power Interface Specification   
***AME***: Attached Matrix Extension   
***ASL***: ACPI Source Language   

### B
***Bias***： 指数偏置。是浮点数表示法中的一个值，用于调整指数范围，使其可以表示负数和正数。在E4M3中，Bias = 2^(4-1)-1 = 7。

### C ###   
***CNNs***: Convolutional Neural Networks 卷积神经网络   
***COFF***: 早期 System V 采用的通用目标文件格式(Common Object File Format)   
***COW***: Copy-On-Write 写时复制   
***CPC***: Continuous Performance Control   
***CPPC***: Collaborative Processor Performance Control   

### D ###
***DLL***: Dynamic Link Library 微软的 Windows 系统中的运行时加载共享库   
***dts***: device tree source 设备树源码是一种描述硬件的数据结构，改变了用hardcode方式将HW配置信息嵌入到内核代码的方法。   
***dwarf***: debug with arbitrary record format   

### E ### 
***ESID***: Exteral Symbol ID 每一个在目标文件中定义或使用的符号都有一个标识符，外部符号标识符。   

### F ###   
***FFH***: Functional Fixed Hardware   
***FIXUPP***: 调整项 重定位项(relocation entry)的别称    

### H ###
***HSM***: Hart State Management   

### I ###
***IME***: Integrated Matrix Extension   

### L ###   
***LPI***: Low Power Idle    

### O ###   
***OCP FP8***: Open Compute Project(OCP)指定的8位浮点数（FP8）规范。E4M3: 1符号、4指数、3位数、指数偏置为7，E5M2: 1符号、5指数、2尾数、指数偏置为15，Exponent、Mantissa。   
***OMF***: Intel Object Module Format Intel 目标模块格式，现存最古老的可执行文件格式之一。   
***OSPM***: Operating System-directed configuration and Power Management    

### P ###
***PIC***: Position Independent Code 将同一段程序加载到不同程序的不同地址的一个常用方法就是位置无关代码   
***PLT***: Procedure Linkage Table 过程链接表   
***PSP***: Program Segment Prefix 程序段前缀。程序入口在0x100处（0-0xFF存放的是程序的命令行参数和其他参数，称为程序段前缀）   

### Q ###
***QMAGIC***: compact pagable format 压缩可分页格式    

### R ###
***RNNs***: Recurrent Neural Networks 循环神经网络   

### S ###   
***SBI***: Supervisor Binary Interface   
***SSA***: static single assignment   
***STL***: standard template library   

### T ###
***TLS***: Thread Local Storage 线程本地存储   

### U ###
***ULP***：units in the last place. or unit of least precision
