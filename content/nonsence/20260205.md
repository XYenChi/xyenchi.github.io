虽然还难以下手支持 RVV，但想到也许我该先测一个 Baseline 来见证 RVV 的负优化。

正常使用在 PyTorch 文件夹路径下使用 `python test/run_test.py --keep-going`即可。   
虽然官方会在缺依赖的时候提醒你使用 .ci 路径下的 requirement.txt 安装依赖，但是某些包（比如 numpy）和本人使用的 openEuler python 3.12 有在 fortran 相关依赖上的冲突。于是选择缺什么安什么。   
大致安装了如下依赖，如果有漏的，也可以顺着网线来找我。   
使用系统包管理器安装 `sudo dnf install python3-scipy`   
使用 pip 在 python 虚拟环境中安装 `pip install expecttest boto3 pytest-rerunfailures pytest-flakefinder`
