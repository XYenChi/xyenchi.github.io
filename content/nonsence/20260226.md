近日为 pytorch riscv 进行测试，过程中有缺少 python 包 monkeytype 的报错，但是 requirements.txt 中没有相关包。   
找到 import monkeytype 的[源码](https://github.com/pytorch/pytorch/blob/main/torch/jit/_monkeytype_config.py)。   
monkeytype 可以通过在运行时监控实际运行的类型信息，自动为现有的 python 代码库生成静态类型注解。   
看起来影响不大，先不管。
